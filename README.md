# Router

Router for retired [Pension Wise] websites.


## Prerequisites

* [Bundler]
* [Git]
* [Ruby 2.2.1][Ruby]


## Installation

Clone the repository:

```sh
$ git clone https://github.com/guidance-guarantee-programme/router.git
```

Resolve dependencies:

```sh
$ bundle
```


## Usage

To start the application:

```sh
$ ./bin/foreman s
```
